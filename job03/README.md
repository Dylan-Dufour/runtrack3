# Taquin — La Plateforme_

Mini-jeu du taquin (8-puzzle) utilisant un logo en image découpée en 9 cases (une vide).

Fichiers créés:
- `index.html` — page principale
- `style.css` — styles
- `script.js` — logique du jeu
- `logo.svg` — placeholder du logo (remplacez par le vrai logo si vous le souhaitez)

Instructions:
1. Ouvrez `index.html` dans un navigateur (double-clic ou via votre serveur local WAMP).
2. Cliquez sur une tuile adjacente à la case vide pour la déplacer.
3. Lorsque l'image est correctement reconstituée, le message "Vous avez gagné" s'affiche en vert et le jeu est bloqué.
4. Cliquez sur "Recommencer" pour lancer une nouvelle partie.

Remarques:
- Le fichier `logo.svg` est un placeholder. Pour utiliser le vrai logo, remplacez ce fichier par l'image du logo (même nom `logo.svg`), ou modifiez la référence dans `style.css`.
- Le mélange est garanti solvable pour le taquin 3x3.
